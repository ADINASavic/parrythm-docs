상세한 설명

시스템 마우스 입력을 Input System 내부의 가상 마우스 장치로 복제하는 컨트롤러다.

실제 하드웨어 마우스 입력을 읽어 VirtualMouse에 반영함으로써, 감도 설정을 가능하게 한다.

입력 흐름: system mouse → virtual mouse → InputSystem → UI / Gameplay
