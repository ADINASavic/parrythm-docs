상세한 설명

런타임 입력 상태를 총괄하는 관리자.

입력 액션 에셋의 복제본을 소유하고, 현재 씬 상태와 임시 override를 바탕으로 어떤 ActionMap 조합을 활성화할지 결정한다.

또한 UI 입력 모듈과 virtual pointer를 동일한 런타임 액션 에셋에 연결한다.
